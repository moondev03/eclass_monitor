# eclass monitor

### 계획
Flutter 토이 프로젝트
비대면 강의 때 사용하던 LMS의 편리함을 eclass에 적용해보고자 함.
과목별 공지, 과제 현황을 보고 알림을 받을 수 있도록 하려고 함.

### 구현 
명지대 통합로그인 후 데이터 스크래핑 (구현 o)
자동로그인 구현 (구현 o)
과목별 과제 필터링 (구현 o)
과목별 과제 진행도 (구현 0)
데이터 새로고침 (구현 o)

과목별 공지 (구현 x)
푸쉬 알림 (구현 x)
...

### 설명
2023년도 2학기 부터 모든 과목 LMS 사용가능으로 변경됨.
따라서 eclass 사용 필요가 없어져 구현 중단함.
